# SuperInstance Fleet Map

An interactive visualization of the SuperInstance fork fleet — 19 open-source projects, 1.4M+ GitHub stars, now conservation-aware.

## What It Shows

- **Stars** sized by parent repo popularity
- **Green glow** = enhanced with a conservation guardian
- **Connections** link repos sharing the same guardian pattern (budget, spectral, size, ecosystem)
- **Hover** for repo details and the key conservation insight

## The Four Guardian Patterns

| Pattern | Color | What It Does |
|---------|-------|-------------|
| Budget Guardian | Green | Tracks resource spend, enforces limits |
| Spectral Analysis | Purple | Reveals hidden structure in graphs/vectors |
| Size/Render | Amber | Optimizes binary size and render performance |
| Ecosystem | Blue | Rust ecosystem intelligence |

## Open It

Just open `index.html` in a browser. No build tools. No dependencies. Just stars.
